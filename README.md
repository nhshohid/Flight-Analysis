# ✈️ Flight Analysis

## Overview

This personal project analyzes over 15.4 million flight records from 2020–2022 to identify trends in delays, cancellations, airline performance, and airport performance. Both static and interactive visualizations are included to highlight key findings.

The main analysis can be found in **Main.ipynb**, while visualizations are located in **Visualizations.ipynb**.

**Note:** Interactive visualizations require cloning the repository and running the notebooks locally in Jupyter Notebook.

## Dataset

Flight Delay Dataset (2020–2022):
https://www.kaggle.com/datasets/robikscube/flight-delay-dataset-20182022

**Note:** Only flight data from 2020-2022 were used. 

## Tools Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Plotly

## Key Findings

* JetBlue Airways recorded the highest average delay time in 2022, at approximately 26 minutes.
* Aspen/Pitkin County Airport (ASE) had the highest average delay time among major airports and a cancellation rate of approximately 9.55%.
* Across all flights analyzed, 3.47% were canceled and 0.22% were diverted.
